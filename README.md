## 📚 Что я изучила

### 1. Что такое алгоритм
**Алгоритм** — это точная последовательность действий для решения задачи.

**5 свойств алгоритма:**
- ✅ Дискретность (состоит из отдельных шагов)
- ✅ Детерминированность (каждый шаг однозначен)
- ✅ Конечность (завершается за конечное время)
- ✅ Массовость (применим для класса задач)
- ✅ Результативность (всегда даёт результат)

### 2. Блок-схемы
Научился(ась) рисовать блок-схемы в **Draw.io** (расширение для VS Code).

**Основные элементы:**
| Форма | Назначение |
|-------|------------|
| 🟠 Овал | Начало/Конец |
| ⬜ Прямоугольник | Действие/Процесс |
| 🔶 Ромб | Условие (Да/Нет) |
| 🔷 Параллелограмм | Ввод/Вывод |
